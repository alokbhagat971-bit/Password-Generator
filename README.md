# 🔐 Password Generator

A simple and beginner-friendly **Password Generator Web App** built using **HTML, CSS, and JavaScript**.
This was one of my early projects while I was starting out in web development, and I’m uploading it now as part of my learning journey 🚀

---

## ✨ Features

* 🔢 Generate passwords of custom length using a slider
* 🎚️ Interactive scroll/slider input for selecting password length
* ⚡ Instant password generation
* 📋 Easy-to-use and clean UI
* 💻 Fully client-side (no backend required)

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling
* **JavaScript** – Logic and interactivity

## 🚀 How to Use

1. Open the app in your browser
2. Use the **slider** to select the desired password length
3. Click the **Generate Password** button
4. Copy and use your secure password
